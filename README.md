# Stock Trading Simulator App

React app made with Hooks and React-Router, that lets user try stock trading simulation with basic tools such as stock quote with graph and latest news, buying, selling and transaction history. Stock data are provided by IEX Cloud API*. Visual styling was done using Material-UI framework.

*- *please note, you would need to use your own API key in order to get IEX Cloud stock data*.

## Tech stack

React, JavaScript, Material-UI, API
